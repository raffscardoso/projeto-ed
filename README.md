## Como Rodar e Compilar

Este projeto utiliza um `Makefile` para automatizar o processo de compilação e execução.

### Comandos do Makefile:

*   **`make`**: Compila o projeto e gera o executável `bin/jogo`.
*   **`make run`**: Compila (se necessário) e executa o programa.
*   **`make clean`**: Remove todos os arquivos temporários (`.o`) e o executável gerado (`bin/jogo`), limpando o diretório do projeto.

### Exemplo de Uso:

Para compilar e executar o jogo:

```bash
make run
```
